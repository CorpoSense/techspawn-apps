SMS Gateway Center

Send SMS using MSG91 and Twilio, Use Templates, Maintain SMS History

To use the Twilio gateway,the User needs to create a Twilio Account and use the given account credentials i.e SID,Authentication Token and Twilio Number. The user needs to install Twilio on their system using sudo apt-get install twilio command.

To use the Msg91 gateway the User needs to create a Msg91 Account and use the given account credentials i.e Authentication key The Sender-Id must be Six Chracter Long without any special chracters.

Send SMS to a single customer

The user can send SMS to a single customer by selecting the customer and using the SEND SMS button located in the Form and Tree view In the send sms form the user must follow the E.164 number format i.e start the mobile number with '+' followed by country code.

Send SMS to Multiple Customers

The user can send SMS to multiple customers at a time.

The Bulk SMS tab can be located under SMS Gateway main menu.

In the send sms form the user must follow the E.164 number format i.e start the mobile number with '+' followed by country code.

The user can select multiple account at a time from the list of accounts added by the user. The user can have multiple accounts with the same SMS Gateway Proider.

Sent SMS History

Different SMS history is maintained for both Multiple customers and Single customer

Resend SMS

The user can resend the sent SMS to the customer whom the message has already been sent by selecting the entry of that sms from the history.

Use predefined templates

The user can create and use the predefined templates and make changes to those templates according to the user requirements.
